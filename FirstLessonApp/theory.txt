КАЖДАЯ СТРОЧКА В ПРОГРАММЕ ЗАКАНЧИВАЕТСЯ ТОЧКОЙ С ЗАПЯТОЙ
Объявление переменной
<тип данных> <идентификатор> [=<начальное значение>];
Типы данных в языке С++
int - целое число 5
float - числа с плавающей точкой 3.5
char - символы 'a'
bool - логический true/false
double - вещественные числа 4.2e-32 15=1.5*10 234=2.34*10^2 4.2*10^-32
void - пустота
Правила задания идентификаторов:
1.Может содержать только латинские буквы, цифры и '_'
2.Не может начинаться с цифры
3.Может начинаться с '_'
4.Не может совпадать с другими идентификаторами
5.Не может совпадать с ключевыми словами языка
6.Регистрозависимы
Примеры объявления переменных:
int i;
float f = 3.5;
char c='a';
bool b = true;
double d = 4.2e-32;
int in=10;
"\n" - переход на новую строку
endl - переход на новую строку
cin >> - оператор ввода
Арифметические операции языка С++:
+ - сложение
- - вычитание
* - умножение
/ - деление
% - остаток от деления
= - присваивание
